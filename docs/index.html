Trial version: user-supplied energies only

Scope of this trial version

This version does not estimate energy values and does not use a database. The user supplies all migration barriers, binding energies, trap depths, and boundary escape barriers in eV. The tool calculates a composition-compatible domain, generates the required energy-input list, runs a trial pathway calculation, and visualises the dominant pathway.
