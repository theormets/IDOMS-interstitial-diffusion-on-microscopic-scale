Interstitial Diffusion on Microscopic Scale — alloy-only graph/KMC trial tool

Revised backend version: user-supplied energies only

Scope

This revised trial version does not estimate energies and does not use a database. The user supplies all site-energy and saddle-energy parameters in eV. The backend constructs the BCC/FCC interstitial graph, assigns site energies and jump barriers, solves first-passage/absorption probabilities, extracts dominant pathways, and returns lattice-aware visualisation data.
